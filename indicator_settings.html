Long Settings

  1. Sensitivity. Allows you to customize the sensitivity of the level search. All for the sake of those people who want aggressive trading to have this choice. The standard sensitivity is Medium. It is suitable for everyone. 
  2. Minimum Touch Count.  Allows you to select which levels to show on the chart, with what minimum number of touches. 2 - standard value.
  3. Bottom Box. It is determined automatically by analyzing volatility, but you can manually specify the boundary of the lower resistance zone in percent.
  4. Top Box. It is determined automatically by analyzing volatility, but you can manually specify the boundary of the upper resistance zone in percent.

Short Settings

Absolutely mirror the Long Settings settings. 

General Settings

  1. Off Long and Off Short Boxes - allow you to disable support or resistance levels (for example, if you consider the indicator only for long trades)
  2. Enable Box Overlap.  Turn on box overlap (Meaning when boxes (support/resistance levels) can be discovered and created inside other boxes).
  3. Smart Box. Experimental setting. If you enable it, there will be much less buy signals, but usually they are as strong as possible and there is a very strong movement when they are broken through.
  4. Max Box Length. You can set the maximum length of the level (support and resistance) by specifying the number of bars. Sometimes it can be useful, but there is no special need for it. 

Design

  1. Allows you to specify colors for everything
  2. Line Style and Line Width.  Allows you to specify the line style and width.
  3. Text in Boxes. If enabled, the prices of the level minimum and maximum will be written inside the levels.
